# Tic-Tac-Toe-Game-in-C-
Tic Tac Toe Game is a console-based C++ application that demonstrates core programming concepts such as arrays, loops, functions, and conditional statements. It features two-player gameplay, dynamic board updates, input validation, win/draw detection, and a replay option, providing an interactive and engaging gaming experience.

## Features
- Two-player mode
- Dynamic game board
- Win detection
- Draw detection
- Replay option
- Input validation

## Technologies Used
- C++
- Arrays
- Loops
- Functions
- Conditional Statements
